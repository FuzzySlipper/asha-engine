## voxel-durability.txt

The saveâ†’compactionâ†’reload durability evidence for the canonical voxel edit sequence:

```text
voxeldurability 1
fixture launch-sequence
postLoad   <hash>   # world fingerprint after the base fixture loads (generation only)
postEdit   <hash>   # after the canonical edit sequence
postReload <hash>   # after compaction + reconstruction
durable    true     # postEdit == postReload
compactedEdits N    retainedEdits M
```

`durable true` is the guarantee: a compacted reload reproduces the edited world exactly.
A tampered snapshot or edit log fails **closed** with a classified `DurabilityError` /
`SnapshotError` rather than loading a divergent world. The TS read model
`@asha/devtools` `buildVoxelDurabilityModel` / `summarizeVoxelDurability` summarizes the
projected form. Background + the deferred generic-`ReplayRecord` unification:
`docs/replay-model.md`.

See also `docs/launchable-voxel.md` for the whole loop.
